Some HTML for Beginners

What is HTML

HyperText Markup Language(HTML) is a computer language that allows one to define the structure and content of a website. It uses tags to define keywords that allows one to build a webpage. There are ten main tags that one must know as an HTML beginner. Thesse tags allow one to create a header, paragraph, block, division, unordered and ordered list, and a link. Furthermore, one can italize text, bold text, and insert an image.

How HTML Works

HTML, which consists of a series of short codes, works through the terminal of ones computer. Through the terminal, you can edit the HTML code for the website stored in their public_html directory. After you finish editing the HTML code, you must write and quit from the vi editor. Then one can view the changes to their website on the browser.

Vi Editor Explained

The Vi Editor allows you access to the HTML code of your website stored in your public_html directory in your local terminal. To access the vi editor you simply type: vi name.html, name being the name of the HTML file. After entering the vi editor you have to simply type, i, for insert, to edit the code. When you are done editing, press the escape button on your keyboard, then shift colon and type w,q. W stands for write and Q stands for quit so you can quit the vi editor.
